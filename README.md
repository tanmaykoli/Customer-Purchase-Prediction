# 📊 Customer Purchase Prediction & Insights Dashboard

## 🚀 Project Overview
This project predicts whether a customer will make a purchase based on demographic and behavioral data.

## 🎯 Objective
- Predict customer purchase behavior
- Analyze key factors influencing buying decisions
- Generate actionable business insights

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 📂 Dataset Features
- Age
- Gender
- Annual Income
- Spending Score
- Purchased (Target)

## 🤖 Models Used
- Logistic Regression
- Random Forest Classifier

## 📊 Results
- Logistic Regression Accuracy: 89%
- Random Forest Accuracy: 100%

## 🔍 Key Insights
- Spending Score and Annual Income are most important
- High-income, high-spending customers are more likely to purchase

## 📈 Business Impact
- Helps businesses target potential customers
- Improves marketing efficiency

## 📌 Future Scope
- Hyperparameter tuning
- Model deployment
- Real-time prediction system
